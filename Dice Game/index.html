Game Instructions

  • The game will start with player one.
  • Player one will roll the dice and the dice number will be added to current score.
  • If the dice number is one then all the current score of the player will be cancelled.
  • If the player click on the hold button then the current score will be added to the actual score and the turn will be give to the next player.
  • The player whose actual score reacher 100 will win the game.
